Description:

My application is going to serve as a manager / companion app for my freelancing business. I will be able to input expenses and incomes and view them as graphs, manage important tasks, export charts, and more.

I will be using Claude to iterate on and design a functional application which I will be able to use outside of class to manage my sole-proprietor business. My plan is to step-by-step begin implementing basic features and once they are built and running I will build upon it with search, visualize and export features.

Planned Features:
- CRUD Expense items
- CRUD Income items
- Search and sort income & expenses
- Visualize data via graphs and charts
- Export data as spreadsheet .csv files

Tech stack:
Vite, React, React DOM, React Router, Tailwind CSS, PostCSS, Autoprefixer
